# smart-traffic-light

create a traffic light management system using machine learning that detects the number of cars on the road using vamera and allocates the sufficient amount of time for the number of cars on the road. train the model using datasets on the internet

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/smart-traffic-light.git
cd smart-traffic-light
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
